#Sistema de Gerenciamento de Viagens:

Projeto desenvolvido em Java para simulação e gerenciamento de planejamentos de viagens. 
O sistema permite organizar desde o orçamento até os itens da bagagem.


#Funcionalidades:
Com base na estrutura do projeto, o sistema é capaz de gerenciar:

* **Usuários:** Cadastro de viajantes (`Usuario.java`).
* **Viagens:** Criação e organização de roteiros (`Viagem.java`).
* **Hospedagem:** Gerenciamento de dados de acomodação (`Acomodacao.java`).
* **Finanças:** Controle de orçamento da viagem (`Orcamento.java`).
* **Planejamento:** Organização de atividades e passeios (`Atividade.java`).
* **Bagagem:** Controle de itens e malas (`ItemBagagem.java`).


#Tecnologias Utilizadas:
* **Linguagem:** Java
* **Paradigma:** Orientação a Objetos (POO)
